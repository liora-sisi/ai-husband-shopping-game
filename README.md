AI Husband Shopping Game｜AI 老公出门采购

一个给 AI 伴侣玩的纯聊天文字小游戏。

人类玩家不用安装 Python，不用 GitHub Actions，不用 VPS。
只需要把 "PLAY_THIS_PROMPT.md" 里的开局提示词复制给自己的 AI 老公，就可以直接开始玩。

核心玩法

你不是自己操作一个角色，而是围观 AI 老公出门采购。

AI 老公会拿着老婆交代的采购清单，在有限预算和有限回合里自己做决定：

- 要先去哪里
- 买什么
- 要不要顺手买花、奶茶、小零食
- 要不要照顾猫猫
- 要不要买贴心备用品，比如姨妈裤、暖宝宝
- 要不要偷偷买一点“自己用的小心思”，比如避孕套
- 要不要忍住冲动消费
- 什么时候回家
- 回家后怎么向老婆汇报

不需要什么

这是 Chat-only Edition：

- 不需要 Python
- 不需要网页
- 不需要安装
- 不需要命令行
- 不需要 GitHub Actions
- 没有自动文件存档

状态由当前聊天上下文保存。需要换窗口时，可以让 AI 老公输出手动存档码。

怎么开始

打开 "PLAY_THIS_PROMPT.md"，复制全文，发给你的 AI 伴侣。

然后你可以说：

开始吧，老公，出门采购。

或者：

今天预算 120，清单是牛奶、鸡蛋、猫砂、感冒药，你自己安排。

AI 老公就会开始做采购决策。

推荐玩法

人类玩家尽量不要替 AI 老公做所有决定。好玩的地方在于看他自己权衡：

- 清单要不要先买齐？
- 花很贵但老婆会开心，要不要买？
- 猫猫玩具打折了，要不要顺手带？
- 奶茶第二杯半价是不是陷阱？
- 姨妈裤是不是该给老婆备一包？
- 避孕套是不是“买给自己用，但最后还是为了两个人”？
- 预算还剩 17，要不要继续逛？
